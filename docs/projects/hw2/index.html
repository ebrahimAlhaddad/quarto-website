Replication of Card and Krueger (1994)

Minimum Wages and Employment: A Case Study of the Fast-Food Industry in New Jersey and Pennsylvania

Author

Ebrahim Alhaddad

Published

May 10, 2026

1 Introduction

In this post I replicate core results from Card and Krueger (1994), a classic paper in empirical economics. The paper studies the effect of New Jersey’s April 1992 minimum wage increase from $4.25 to $5.05 per hour on employment in fast-food restaurants. The authors compare restaurants in New Jersey, which received the policy change, to restaurants in eastern Pennsylvania, where the minimum wage did not change. This gives a natural difference-in-differences design.

The paper focuses on a simple question: did the higher minimum wage reduce employment? Under the standard competitive model, the answer should be yes. But Card and Krueger find no evidence of job loss. In fact, their data suggest that employment in New Jersey rose relative to Pennsylvania.

For this assignment I do three things:

  1. Replicate the key descriptive comparison in Table 2.
  2. Replicate the core difference-in-differences result from Table 3 and the basic regression from Table 4, column (i).
  3. Add one extra piece of analysis by graphing the before/after employment patterns that the paper mostly presents in table form.

2 Main Analysis

2.1 Load and prepare the data

The public data are stored in a fixed-width ASCII file (public.dat). I use the codebook and the original SAS checking program to reconstruct the variables used in the paper, including:

  • FTE employment = full-time employees + managers + 0.5 × part-time employees
  • gap = proportional wage increase needed to reach the new minimum wage in New Jersey
  • pmeal = price of a full meal = soda + fries + entree
  • indicators for chain type and New Jersey vs. Pennsylvania

df = raw.copy()

# Construct variables used in the paper / SAS check file
# Full-time-equivalent employment

df["emptot"] = df["empft"] + df["nmgrs"] + 0.5 * df["emppt"]
df["emptot2"] = df["empft2"] + df["nmgrs2"] + 0.5 * df["emppt2"]
df["demp"] = df["emptot2"] - df["emptot"]

df["gap"] = np.where(
    df["state"] == 0,
    0,
    np.where(
        df["wage_st"] >= 5.05,
        0,
        np.where(df["wage_st"] > 0, (5.05 - df["wage_st"]) / df["wage_st"], np.nan)
    )
)

# State and chain indicators

df["nj"] = df["state"]
df["bk"] = (df["chain"] == 1).astype(int)
df["kfc"] = (df["chain"] == 2).astype(int)
df["roys"] = (df["chain"] == 3).astype(int)
df["wendys"] = (df["chain"] == 4).astype(int)

# Other variables used in tables

df["pmeal"] = df["psoda"] + df["pfry"] + df["pentree"]
df["pmeal2"] = df["psoda2"] + df["pfry2"] + df["pentree2"]
df["fracft"] = df["empft"] / df["emptot"]
df["fracft2"] = np.where(df["emptot2"] > 0, df["empft2"] / df["emptot2"], np.nan)
df["atmin"] = (df["wage_st"] == 4.25).astype(int)
df["newmin"] = (df["wage_st2"] == 5.05).astype(int)
df["closed"] = (df["status2"] == 3).astype(int)

# The paper treats the four temporarily closed stores as zero only in one row of Table 3.
# These statuses correspond to renovation / temporary closures / fire / construction.
TEMP_CLOSED = {2, 4, 5}
df["emptot2_zero_temp"] = df["emptot2"]
df.loc[df["status2"].isin(TEMP_CLOSED), "emptot2_zero_temp"] = 0
df["demp_zero_temp"] = df["emptot2_zero_temp"] - df["emptot"]

# Samples used below
balanced = df[df["demp"].notna()].copy()
row5_sample = df[(df["demp"].notna()) | (df["status2"].isin(TEMP_CLOSED) & df["emptot"].notna())].copy()

def mean_se(x):
    x = pd.to_numeric(x, errors="coerce").dropna()
    return pd.Series({"mean": x.mean(), "se": x.std(ddof=1) / np.sqrt(len(x)), "n": len(x)})

2.2 Replicating Table 2

Table 2 compares New Jersey and Pennsylvania stores before and after the policy change. My replicated means line up closely with the paper. The two most important patterns are the same as in the published table:

  • In wave 1, average FTE employment is lower in New Jersey than in Pennsylvania (about 20.44 vs. 23.33).
  • In wave 2, New Jersey’s average starting wage rises sharply to about 5.08, while Pennsylvania stays near 4.62.
def t_stat(var):

2.3 Replicating Table 3: the key difference-in-differences result

Table 3 is the heart of the paper. The key row is the balanced sample comparison, where employment is observed in both periods.

The difference-in-differences estimate is:

[ (Employment_{NJ}) - (Employment_{PA}) ]

Using the balanced sample, I get:

  • Pennsylvania change: (-2.28) FTE workers
  • New Jersey change: (+0.47) FTE workers
  • Difference-in-differences: (+2.75) FTE workers

This essentially matches the paper’s reported +2.76 result.

pa = df[df["nj"] == 0]
nj = df[df["nj"] == 1]
low = df[(df["nj"] == 1) & (df["wage_st"] == 4.25)]
mid = df[(df["nj"] == 1) & (df["wage_st"] > 4.25) & (df["wage_st"] < 5.00)]
high = df[(df["nj"] == 1) & (df["wage_st"] >= 5.00)]

2.4 Replicating Table 4, column (i)

Table 4 turns the simple comparison into a regression. The required replication is column (i), where the dependent variable is the change in FTE employment and the key regressor is a New Jersey dummy.

Following the paper, I restrict to the subset of stores with available employment and wage data in both waves (or stores that closed in wave 2). This yields the same sample size as the paper: 357 stores.

# Same sample restriction as the SAS check program / Table 4
reg_sample = df[df["demp"].notna() & ((df["closed"] == 1) | (df["wage_st2"] - df["wage_st"]).notna())].copy()

m1 = smf.ols("demp ~ nj", data=reg_sample).fit()
m2 = smf.ols("demp ~ nj + bk + kfc + roys + co_owned", data=reg_sample).fit()
m3 = smf.ols("demp ~ gap", data=reg_sample).fit()
m4 = smf.ols("demp ~ gap + bk + kfc + roys + co_owned", data=reg_sample).fit()
m5 = smf.ols("demp ~ gap + bk + kfc + roys + centralj + southj + pa1 + pa2", data=reg_sample).fit()

3 Something Additional

For my additional analysis, I turn the paper’s table-based comparison into a picture. The paper discusses these patterns in Table 3, but a graph makes the result much easier to see.

I plot average FTE employment before and after the policy for four groups:

  • Pennsylvania stores
  • all New Jersey stores
  • low-wage New Jersey stores
  • high-wage New Jersey stores
plot_df = pd.DataFrame({
    "group": ["Pennsylvania", "New Jersey", "NJ low wage", "NJ high wage"],
    "Before": [pa["emptot"].mean(), nj["emptot"].mean(), low["emptot"].mean(), high["emptot"].mean()],
    "After": [pa["emptot2"].mean(), nj["emptot2"].mean(), low["emptot2"].mean(), high["emptot2"].mean()],
})

fig, ax = plt.subplots(figsize=(8, 5))
for _, row in plot_df.iterrows():
    ax.plot(["Before", "After"], [row["Before"], row["After"]], marker="o", label=row["group"])

ax.set_title("Average FTE Employment Before and After the 1992 Minimum Wage Increase")
ax.set_ylabel("Average FTE employment")
ax.legend(frameon=False)
plt.show()

The figure highlights the same substantive point as the tables:

  • Pennsylvania stores lost workers over time.
  • New Jersey stores, on average, did not.
  • The strongest relative increase comes from the low-wage New Jersey stores, which were the stores most directly affected by the policy.

That visual pattern supports the paper’s interpretation that the minimum wage increase did not reduce employment in this setting.

4 Conclusion

The results here line up closely with the original paper.

I reproduced the main findings from Card and Krueger (1994) using only Python and the public fast-food survey data:

  • The descriptive means in Table 2 match the paper closely.
  • The balanced-sample difference-in-differences estimate from Table 3 is about +2.75 FTE workers, essentially identical to the paper’s +2.76.
  • The regression estimate from Table 4, column (i) is 2.33 with a standard error of 1.19, which matches the published result.

Overall, the replicated evidence supports the paper’s headline conclusion: the 1992 New Jersey minimum wage increase did not reduce fast-food employment relative to Pennsylvania.
